# E/PRO (Backend API)
This is the backend API for the E/PRO mobile application.

## Built With
- Bcrypt
- Express.js
- PostgreSQL

## Routes
- /users
- /users/:id
- /users/trainers
- /users/trainers/:id
- /users/:id/workouts
- /users/:id/workouts/:date
- /exercises/
- /exercises/:id
- /exercises/:workout_exercise_id/:exercise_id/
- /phase_tips/
- /phase_tips/:id
- /users/register
- /auth/login
- /auth/status
- /auth/logout
- /hormones/non_hormonal
- /hormones/triphasic
- /hormones/monophasic
- /hormones/progestin
- /hormones/non_hormonal/:id
- /hormones/triphasic/:id
- /hormones/monophasic/:id
- /hormones/progestin/:id
